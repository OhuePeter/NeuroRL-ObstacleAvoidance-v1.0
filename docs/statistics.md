# Statistical Analysis

The following behavioural metrics will be computed.

## Behaviour

- Success Rate
- Collision Rate
- Timeout Rate
- Time to Goal
- Route Selection
- Path Length
- Velocity
- Acceleration
- Curvature
- Recovery Time
- Maximum Deviation

## Neural Population

- PCA
- Explained Variance
- Population Norm
- Correlation Matrix
- Hierarchical Clustering
- Decoder Accuracy
- Cluster Statistics

Comparisons will be performed between

- Unperturbed
- Small Perturbation
- Medium Perturbation
- Large Perturbation

for both left and right perturbations.
